Formatação HTML

O HTML contém vários elementos para definir o texto com um significado especial.

Exemplo

Esse texto está em negrito

Esse texto está em itálico

Esse é um texto subscrito e sobrescrito

Código

<p><b> Esse texto está em negrito </b></p>

<p><i> Esse texto está em itálico </i></p>

<p> Esse é um texto <sub> subscrito </sub> e <sup> sobrescrito </sup></p>

Elementos de formatação HTML

Os elementos de formatação foram projetados para exibir tipos especiais de texto:

  • <b> Texto em negrito
  • <strong> Texto importante
  • <i> Texto em itálico
  • <em> Texto enfatizado
  • <mark> Texto marcado
  • <small> Texto pequeno
  • <del> Texto excluído
  • <ins> Texto inserido
  • <sub> Texto subscrito
  • <sup> Texto sobrescrito

Elementos <b> e <strong>

O elemento <b> define um texto em negrito, sem muita importância.

Exemplo

<b>Esse texto está em negrito</b>

Resultado

Esse texto está em negrito

O elemento <strong> define o texto com grande importância. O conteúdo interno é normalmente exibido em negrito.

Exemplo

<strong>Esse texto é importante!</strong>

Resultado

Esse texto é importante!

Elementos <i> e <em>

O elemento <i> define uma parte do texto em uma voz ou humor alternativo. O conteúdo interno é normalmente exibido em itálico.

Dica: A tag <i> é frequentemente usada para indicar um termo técnico, uma frase de outro idioma um pensamento, um nome de um navio, etc.

Exemplo

<i>Esse texto está em itálico</i>

Resultado

Esse texto está em itálico

O elemento <em> define o texto enfatizado. O conteúdo interno é exibido normalmente em itálico.

Dica: um leitor de tela pronunciará as palavras em <em> com ênfase, usando ênfase verbal.

Exemplo

<em>Esse texto está enfatizado</em>

Resultado

Esse texto está enfatizado

Elemento <small>

O elemento <small> define um texto pequeno:

Exemplo

<small>Esse é um texto pequeno</small>

Resultado

Esse é um texto menor.

Elemento <mark>

O elemento <mark> define o texto que deve ser marcado ou destacado:

Exemplo

<p>Do not forget to buy <mark>milk</mark> today.</p>

Resultado

Do not forget to buy milk today.

Elemento <del>

O elemento <del> define o texto que foi excluído de um documento. Os navegadores geralmente riscarão uma linha no texto excluído.

Exemplo

<p>Minha cor favorita é <del> azul </del> vermelho.</p>

Resultado

Minha cor favorita é azul vermelho.

Elemento <ins>

O elemento <ins> define um texto que foi inserido em um documento. Os navegadores geralmente sublinham o texto inserido.

Exemplo

<p>Minha cor favorita é <del>azul</del><ins>vermelho.</ins>

Resultado

Minha cor favorita é azul vermelho.

Elemento <sub>

O elemento <sub> define o texto subscrito. O texto subscrito aparece meio caractere abaixo da linha normal e, as vezes, é renderizado em uma fonte menor. O texto subscrito pode ser usado para fórmulas químicas como H2O:

Exemplo

<p>Esse é um texto <sub>subscrito.</sub></p>

Resultado

Esse é um texto subscrito.

Elemento <sup>

O elemento <sup> define o texto sobrescrito. O texto sobrescrito aparece com meio caractere acima da linha normal e, as vezes, é renderizado com uma fonte menor. Texto sobrescrito pode ser usado para notas de rodapé, como WWW[1]

Exemplo

<p>Esse é um texto <sup>sobrescrito</sup></p>

Resultado

Esse é um texto sobrescrito.
