Usando <blockquote> para quotações em HTML

O elemento HTML <blockquote> define uma seção que é citada de outra fonte.

Os navegadores geralmente recuam elementos <blockquote>.

Exemplo

<p>Aqui está uma citação do site Wikipedia:</p>

<blockquote cite="https://pt.wikipedia.org/wiki/Hist%C3%B3ria_da_Internet">

A ARPANET funcionava através de um sistema conhecido como chaveamento de pacotes, que é um sistema de transmissão de dados em rede de computadores no qual as informações são divididas em pequenos pacotes, que por sua vez contém trecho dos dados, o endereço do destinatário e informações que permitiam a remontagem da mensagem original.

</blockquote>

Clique aqui para ver o resultado

Aqui está uma citação do site Wikipedia:

A ARPANET funcionava através de um sistema conhecido como chaveamento de pacotes, que é um sistema de transmissão de dados em rede de computadores no qual as informações são divididas em pequenos pacotes, que por sua vez contém trecho dos dados, o endereço do destinatário e informações que permitiam a remontagem da mensagem original.

Use a tag HTML <q> para citações curtas

A tag HTML <q> define uma quotação curta.

Os navegadores normalmente inserem aspas ao redor da cotação.

Exemplo

<p><q>Já na década de 1970, a tensão entre URSS e EUA diminui.<q><p>
Clique aqui para ver o resultado

Já na década de 1970, a tensão entre URSS e EUA diminui.


Usando a tag <abbr> para abreviações

A tag HTML define uma abreviação ou um acrônimo, como "HTML", "CSS", "Mr.", "Dr.", "ASAP", "ATM".

A marcação de abreviaturas pode fornecer informações úteis para navegadores, sistemas de tradução e mecanismos de busca.

Dica: Use o atributo global title para mostrar a descrição da abreviação/acrônimo ao passar o mouse sobre o elemento.

Exemplo

<p>A <abbr title="Advanced Research Projects Agency">ARPANET</abbr> foi criada em 1962.</p>
Clique aqui para ver o resultado

A ARPANET foi criada em 1962.

Usando a tag <address> para informações de contato

A tag HTML <address> define as informações de contato do autor/proprietário de um documento ou artigo.

As informações de contato podem ser um endereço de e-mail, URL, endereço físico, número de telefone, identificador de mídia social, etc.

O texto no elemento <address> geralmente é renderizado em itálico, e os navegadores sempre adicionam uma quebra de linha antes e depois do elemento <address>.

Exemplo

<address>

Escrito por campos-gustavo.<br>

Visite-nos em:<br>

Exemplo.com.br<br>

Box 564, Disneylandia<br>

EUA

</address>

Clique aqui para ver o resultado

Exemplo.com.br
Box 564, Disneylandia
EUA

Elemento <cite> para títulos de trabalho

A tag HTML <cite> define o título de um trabalho criativo(por exemplo, um livro, um poema, uma música, um filme, uma pintura, uma escultura, etc.).

Nota:O nome de uma pessoa não deve fazer parte do título de uma obra.

O texto no elemento <cite> geralmente é renderizado em itálico.

Exemplo

<p><cite>The Scream</cite> by Edvard Munch. Painted in 1893.</p>
Clique aqui para ver o resultado

The Scream by Edvard Munch. Painted in 1893.

Elemento <bdo> para substituição biderecional

BDO signifca stands for Bi-Directional Override.

A tag HTML <bdo> é usada para substituir a direção do texto atual:

Exemplo

<bdo dir="rtl">Esse texto vai ser escrito da direita para a esquerda
Clique aqui para ver o resultado
Esse texto vai ser escrito da direita para a esquerda

Quotações HTML e elementos de citação

tag Description
<abbr> Define uma abreviação ou um acronimo
<address> Define uma informação de contato para o autor/dono do documento
<bdo> Define a direção do texto
<blockquote> Define uma seção que é citada de outra fonte
<cite> Define o título de um trabalho
<q> Define uma cotação em uma linha curta
