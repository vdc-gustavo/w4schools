Um editor de texto simples é tudo que você precisa para aprender HTML.

Aprenda HTML usando Notepad++ ou TextEdit

As páginas Web podem ser criadas e modificadas usando editores de HTML profissionais.

No entanto, para aprender HTML, recomendamos um editor de texto simples como o Notepad++ (Windows) ou TextEdit(Mac).

Acreditamos que usar um editor de texto simples é uma boa maneira de aprender HTML.

Siga as etapas abaixo para criar sua primeira página Web com o Notepad++ ou TextEdit.

Etapa 1: abra seu editor de texto

Etapa 2: escreva um pouco de HTML

Escreva ou copie o seguinte código no editor de texto:

<!DOCTYPE html>

<html>

<body>

<h1> Meu primeiro título </h1>

<p> Meu primeiro parágrafo. </p>

</body>

</html>

Etapa 3: Salve a página HTML

Salve o arquivo no computador. Selecione Arquivo > Salvar como no menu do editor.

Nomeie o arquivo para "index.html".

Nota: você pode usar .htm ou .html como extensão do arquivo. não há diferença, cabe a você escolher.

Etapa 4: Veja a página HTML no seu navegador.

Abra o arquivo HTML salvo em seu navegador favorito (clique duas vezes no arquivo ou clique com o botão direito e escolha "Abrir com").
