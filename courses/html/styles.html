Estilos HTML

O atributo <style> é usado para adicionar estilos a um elemento, como cor, fonte, tamanho e muito mais.

Exemplo

Eu sou vermelho

Eu sou azul

Eu sou GRANDE

O atributo style

Definir o estilo de um elemento HTML pode ser feito com o atributo style.

O atributo de estilo HTML possui a seguinte sintaxe:

Exemplo

<tagname style="property:value;">

property é uma propriedade CSS.
value é um valor CSS.

Background Color

A propriedade CSS background-color define a cor de fundo para um elemento HTML.

Exemplo

Define a cor de fundo da página para powderblue:

<body style="background-color:powderblue;">

<h1>Isso é um cabeçalho</h1>

<p>Isso é um parágrafo</p>

</body>

Exemplo

<body>

<h1 style="background-color:powderblue;">Isso é um cabeçalho</h1>

<p style="background-color:tomato;">Isso é um parágrafo</p>

Cor do texto

A propriedade CSS color define a cor de um texto para um elemento HTML:

Exemplo

<h1 style="color:blue";>Isso é um cabeçalho</h1>

<p style="color:red";>Isso é um parágrafo.</p>

Fontes

A propriedade CSS font-family define qual a fonte a ser usado no elemento HTML

Exemplo

<h1 style="font-family:verdana;">Isso é um cabeçalho</h1>

<p style="font-family:courier;">Isso é um parágrafo.</p>

Tamanho de texto

A propriedade CSS font-size define o tamanho da fonte para um elemento HTML

Exemplo

<h1 style="font-size:300%;">Isso é um cabeçalho</h1>

<p style="font-size:160%">Isso é um parágrafo.</p>

Alinhamento de texto

A propriedade CSS text-align define o alinhamento horizontal do texto de um elemento HTML

Exemplo

<h1 style="text-align:center;">Cabeçalho centralizado</h1>

<p style="text-align:center;">Parágrafo centralizado.</p>
