Parágrafos HTML

Um parágrafo sempre começa em uma nova linha e geralmente é um bloco de texto.

Parágrafos HTML

A tag <p> define um parágrafo.

Um parágrafo sempre começa em uma nova linha e os navegadores adicionam automaticamente algum espaço em branco (uma margem) antes e depois de um parágrafo.

Exemplo

<p>Isso é um parágrafo.</p>

<p>Isso é um outro parágrafo.</p>

Display HTML

Você não pode ter certeza de como o HTML será exibido.

Telas grandes ou pequenas e janelas redimensionadas criarão resultados diferentes.

Com HTML, você não pode alterar a exibição adicionando espaços extras ou linhas extras em seu código HTML.

O navegador removerá automaticamente quaisquer espaços e linhas extras quando a página for exibida:

Exemplo

<p>
Esse
parágrafo
contém
muitas
linhas
no
código,
mas
o
navegador
ignora
isso.
</p>

Regras horizontais de HTML

A tag <hr> define uma quebra temática em uma página HTML e geralmente é exibida como uma régua horizontal.

O elemento <hr> é usado para separar o conteúdo (ou definir uma mudança) em uma página HTML:

Exemplo

<h1>Isso é um cabeçalho de nível 1</h1>

<p>Isso é um pouco de texto</p>

<hr>

<h2>Isso é um cabeçalho de nível 2</h2>

<p>Isso é um outro texto</p>

<hr>

A tag <hr> é uma tag vazia, o que significa que não possui tag final.

Quebras de linha HTML

A tag <br> define uma quebra de linha no texto

Use <br> se quiser uma quebra de linha (uma nova linha) sem iniciar um novo parágrafo:

Exemplo

<p>Isso é <br>um parágrafo <br> com quebras de linha.</p>

A tag <br> é uma tag vazia, o que significa que não possui tag final.

O problema do poema

Esse poema vai ser mostrado em uma única linha:

Exemplo

<p>

Minha Bonnie está deitada sobre o oceano.

Minha Bonnie está deitada sobre o mar.

Minha Bonnie está deitada sobre o oceano.

Oh, traga minha Bonnie de volta para mim.

</p>

Solução - O elemento <pre>

O elemento HTML <pre> define o texto pré-formatado.

O texto dentro de um elemento <pre> é exibido em uma fonte de largura fixa (geralmente Courier) e preserva espaços e quebras de linha:

<pre>

Minha Bonnie está deitada sobre o oceano.

Minha Bonnie está deitada sobre o mar.

Minha Bonnie está deitada sobre o oceano.

Oh, traga minha Bonnie de volta para mim.

</pre>
