Exemplos básicos de HTML

Neste capítulo, mostraremos alguns exemplos básicos de HTML.

Não se preocupe se usarmos tags que você ainda não conhece.

Documentos HTML

Todos os documentos HTML devem começar com uma declaração de tipo de documento: <!DOCTYPE html> .

O próprio documento HTML começa com <html> e termina com </html>.

A parte visível do documento HTML está entre<body> e </body>.

Exemplo

<!DOCTYPE html>

<html>

<body>

<h1>Meu primeiro título</h1>

<p>Meu primeiro parágrafo</p>

</body>

</html>

Resultado

Meu primeiro título

Meu primeiro parágrafo

A declaração <!DOCTYPE>

A declaração <!DOCTYPE> representa o tipo de documento e ajuda os navegadores a exibir as páginas da web corretamente.

Deve aparecer apenas uma vez, no topo da página (antes de quaisquer tags HTML).

A declaração <!DOCTYPE> não faz distinção entre maiúsculas e minúsculas.

A declaração <!DOCTYPE> para HTML5 é:

<!DOCTYPE html>

Cabeçalhos HTML

Os cabeçalhos HTML são definidos com as tags <h1> a <h6>.

<h1> define o cabeçalho de nível maior. <h6> define o cabeçalho de nível menor

Exemplo

<h1>Isso é um cabeçalho de nível 1</h1>

<h2>Isso é um cabeçalho de nível 2</h2>

<h3>Isso é um cabeçalho de nível 3</h3>

Resultado

Isso é um cabeçalho de nível 1

Isso é um cabeçalho de nível 2

Isso é um cabeçalho de nível 3

Parágrafos em HTML

Parágrafos em HTML são definidos com a tag <p>

Exemplos

<p>Isso é um parágrafo</p>

<p>Isso é outro parágrafo</p>

Resultado

Isso é um parágrafo

Isso é outro parágrafo

Links em HTML

Links em HTML são definidos com a tag <a>

Exemplo

<a href="G3XiSchools.com">Isso é um link</a>

Resultado
Isso é um link

O destino do link é especificado no atributo href.

Os atributos são usados ​​para fornecer informações adicionais sobre os elementos HTML.

Você aprenderá mais sobre atributos em um capítulo posterior.

Imagens em HTML

Imagens em HTML são definidas com a tag <img>

O arquivo de origem (src), texto alternativo (alt), largura e altura são fornecidos como atributos:

Exemplo

<img src="unknown.png" alt="unknown" width="104" height="142">

Resultado

Como visualizar o código fonte HTML da página?

Você já viu uma página da Web e se perguntou "Ei! Como eles fizeram isso?"

Ver código fonte da página

Clique com o botão direito em uma página HTML e selecione "Exibir código fonte da página" (no Opera), "Visualizar Fonte da Página" (no Chrome) ou "Ver origem" (no Edge), etc. Isso abrirá uma nova aba contendo o código fonte HTML da página.

Inspecionar um elemento HTML

Clique com o botão direito do mouse em um elemento (ou em uma área em branco) e escolha "Inspecionar" ou "Inspecionar elemento" para ver quais elementos são compostos (você verá o HTML e as CSS). Você também pode editar o HTML ou CSS no painel que se abrir.
