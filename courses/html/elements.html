Um elemento HTML é definido por uma tag inicial, algum conteúdo e uma tag final.

Elementos HTML

O elemento HTML é tudo desde a tag inicial até a tag final:

<tag>Conteúdo vem aqui...</tag>

Exemplos de alguns elementos HTML:

<h1>Meu primeiro título</h1>

<p>Meu primeiro parágrafo</p>

exemplo incompleto (tabelas)

Nota: alguns elementos HTML não têm conteúdo (como o elemento <br>). Esses elementos são chamados de elementos vazios. Os elementos vazios não possuem uma tag de finalização!

Elementos HTML aninhados

Os elementos HTML podem ser aninhados (isso significa que os elementos podem conter outros elementos).

Todos os documentos HTML consistem em elementos HTML aninhados.

O exemplo a seguir contém quatro elementos HTML (<html>, <body>, <h1> e <p>):

Exemplo

<!DOCTYPE html>

<html>

<body>

<h1>Meu primeiro título</h1>

<p>Meu primeiro parágrafo</p>

<body>

<html>

Resultado

Meu primeiro título

Meu primeiro parágrafo.

Explicando o exemplo

O elemento <html> é o elemento raiz e define todo o documento HTML.

Possui uma tag inicial <html> e uma tag final </html>.

Então, dentro do elemento <html>, há um elemento <body>:

<body>

<h1>Meu primeiro título</h1>

<p>Meu primeiro parágrafo</p>

</body>

O elemento <body> define o corpo do documento

Possui uma tag inicial <body> e uma tag final </body>.

Então, dentro do elemento <body>, existem dois outros elementos: <h1> e <p>:

<h1>Meu primeiro título</h1>

<p>Meu primeiro parágrafo</p>

O elemento <h1> define um cabeçalho de nível 1

Possui uma tag inicial <h1> e uma tag final &mei/h1>.

<h1>Meu primeiro título</h1>

O elemento <p> define um parágrafo.

Possui uma tag inicial <p> e uma tag final </p>.

<p>Meu primeiro parágrafo</p>



Nunca pule a tag final

Alguns elementos HTML serão exibidos corretamente, mesmo se você esquecer a tag final:

Exemplo

<html>

<body>

<p>Isso é um parágrafo

<p>Isso é um parágrafo

</body>

</html>

Resultado

Isso é um parágrafo

Isso é um parágrafo

No entanto, nunca confie nisso! Resultados inesperados e erros podem ocorrer se você esquecer a tag final!

Elementos HTML vazios

Os elementos sem conteúdo são chamados de elementos vazios

A tag <br> define uma quebra de linha, e é um elemento vazio sem uma tag final:

Exemplo

<p>Isso é um <br> parágrafo com quebra de linha</p>

Resultado

Isso é um

parágrafo com uma quebra de linha

HTML não diferencia maiúsculas de minúsculas

As tags HTML não diferenciam maiúsculas de minúsculas: <P> significa o mesmo que <p>.

O padrão HTML não requer tags em minúsculas, mas recomendamos minúsculas em HTML e exige minúsculas para tipos de documentos mais restritos como XHTML.
